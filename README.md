# UniLiving-Space

开发web端，H5端，小程序。提供房源租赁服务

他乡寓（UniLiving Space）是一个结合区块链技术的全球性的留学、差旅找房租房平台。我们通过品牌信息上链、用户分享上链等方式，为用户提供优质公寓查询、租房推荐、酒店预订等服务。目前合作品牌主要分布香港、澳门、伦敦、悉尼等热门留学城市。



目前准备使用的技术栈为：

##### 后端

1. django-rest-frame-work
2. rest-frame-work-cors
3. django-filter



##### 前端

1. Vue3
2. Vuex4
3. Vue-Router

4. axios
